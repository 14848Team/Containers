The floder "venv" is the virtual environment which includes all the python libraries used in this project.
The floder "handout" contains our source code.
The file "part1.txt" includes commands history we have entered to establish a container 

To test rest api:

    1. make manager
    2. make api_tests


To test cli:

    1. make manager 
        (if you don't close the server after running the rest api test, this step can be skipped)
    2. make cli_tests
